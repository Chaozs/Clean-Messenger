# Clean-Messenger


## Required dependencies:

### python3:
sudo apt-get install python3.6

### pip
sudo apt install python3-pip

### cython
pip install Cython

### word2vec
pip install word2vec

### Levenshtein library
pip install python-Levenshtein

### Tkinter library
sudo apt-get install python3-tk


## To Run project:

### Start server:
python server.py

### Start client:
python server.py
